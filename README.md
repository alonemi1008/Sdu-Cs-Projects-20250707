## 项目列表

### Project1-SM4
SM4分组密码算法的三种优化实现：
- 查表优化（Table-based）
- SIMD向量指令集优化
- AES-NI指令集优化

详细说明请见[Project1-SM4/readme.md](./Project1-SM4/readme.md)

## 性能对比

| 优化方式 | 加密性能 | 特点 |
|---------|---------|------|
| 查表优化 | 中等 | 通过预计算S盒和T变换提高效率 |
| SIMD指令集 | 较高 | 使用向量指令并行处理多组数据 |
| AES-NI指令集 | 最高 | 利用Intel专用硬件加速指令 |

## 环境要求
- 支持C++11标准
- 对于SIMD优化，需要支持SSE/AVX指令集
- 对于AES-NI优化，需要Intel处理器并支持AES-NI指令集
